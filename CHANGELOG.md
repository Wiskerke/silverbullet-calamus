# Changelog

## Unreleased

- Widget status bar now is a link that opens the note
- Replaced deprecated `space.readAttachment` with `space.readDocument`

## v0.1.0 — 2026-03-03

Initial release.

- Code widget for `note` fenced blocks: renders pages from a `.note` file inline
- Document editor for `.note` files: full read-only page viewer
- Theme support: configurable light/dark colors, global config via `SETTINGS` and per-widget overrides
- Page selection: single page, range (`1-3`), list (`1,3,5`), or all pages
